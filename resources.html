Resources used in this project

  • Leaflet

    Leaflet is an open source Javascript library used to impliment map containers on a webpage.

  • Garage Availability

    Texas A&M Transportation Services live count of visitor parking garage availability for the different garages on campus.

  • Map Tiler Cloud

    Used to create the GeoJSON of building outlines used on our main map.

  • Alidade Smooth Basemap

    Alidade Smooth by Stadia Maps is the basemap we used in our map display.

  • Google Fonts Oswald

    The Oswald Font by Google Fonts was used for titles on our webpages. Oswald is one of the offical fonts of Texas A&M.

  • Texas A&M Branding Materials

    We used official Aggie branding materials such as fonts, colors, and other design items.

  • ArcGIS Pro

    ArcGIS Pro was used to modify GeoJSON files and add attributes.

  • Google Workspace

    Used Google Forms and Google Sheets to gather and store user submitted data for use on webmap.
